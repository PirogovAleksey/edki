# Мережева безпека

## Модель OSI

Семирівнева модель мережевої взаємодії:

1. **Фізичний** — кабелі, сигнали
2. **Канальний** — MAC-адреси, Ethernet
3. **Мережевий** — IP-адресація, маршрутизація
4. **Транспортний** — TCP/UDP, порти
5. **Сеансовий** — управління з'єднаннями
6. **Представлення** — кодування, шифрування
7. **Прикладний** — HTTP, DNS, SMTP

## Фаєрволи

Контролюють мережевий трафік на основі правил.

- **Пакетний фільтр** — аналізує заголовки пакетів (L3-L4)
- **Stateful** — відстежує стан з'єднань
- **WAF** — захищає веб-застосунки (L7)

## VPN

Створює захищений тунель через незахищену мережу.

- **IPsec** — шифрування на мережевому рівні
- **OpenVPN** — працює через TLS
- **WireGuard** — сучасний, швидкий протокол

## IDS / IPS

- **IDS** — виявляє вторгнення, сповіщає адміністратора
- **IPS** — виявляє та блокує вторгнення автоматично

Типи: мережевий (NIDS/NIPS) та хостовий (HIDS/HIPS).
